Como Funciona:

      O código começa selecionando elementos HTML relevantes para a interação do usuário, como botões de escolha, elementos de texto e elementos de entrada. Isso é feito usando o método document.querySelector() para encontrar elementos com classes específicas ou IDs.
      Em seguida, o código inicializa algumas variáveis, incluindo um contador a e uma flag confirmacao, que é usada para indicar se a resposta do usuário está correta.
      Há duas funções de manipulação de eventos: enviarFormulario() e capturarValor(). A primeira é acionada quando o usuário pressiona a tecla Enter, enquanto a segunda verifica e processa a resposta fornecida pelo usuário.
      Os dados sobre as ilhas são representados através de dois arrays: arrayIlha, que contém os nomes das ilhas, e caminhoilhas, que define as conexões entre elas.
      A função algo() manipula a interação do usuário com as ilhas. Quando o usuário clica em uma ilha, esta função é chamada para atualizar dinamicamente o conteúdo da página com base na ilha selecionada.
      Eventos de clique são adicionados a cada opção de escolha para permitir que o usuário interaja com as ilhas. Isso é feito usando um loop forEach() para iterar sobre os elementos de escolha.
      Finalmente, dois arrays, arrayPerguntas e arrayResposta, são utilizados para armazenar perguntas e respostas correspondentes, respectivamente.
      Em resumo, o código JavaScript fornece um exemplo básico de como criar uma experiência interativa em uma página da web. Ele demonstra técnicas para manipular o DOM e responder às interações do usuário, mostrando como o JavaScript pode ser usado para criar aplicativos dinâmicos e envolventes em um navegador web.
